About This Project

This dashboard showcases my ability to build an end-to-end risk intelligence product: framing a real-world business problem, designing a practical analytics model, and translating findings into an executive-ready interface.

Key Decisions

  • • Prioritized actionable risk signals over vanity metrics
  • • Shifted from state-only view to national synthesis cards
  • • Built data-driven correlation logic for policy/media/social

Tech & Execution

  • • Node.js + Express API layer
  • • Vanilla JS analytics and dynamic rendering
  • • Tailwind-based responsive intelligence dashboard

Lessons Learned

  • • Data quality and query scope define insight quality
  • • Executive Summary UX needs clarity, hierarchy, and context
  • • Correlated signals outperform isolated indicators
  • • Multi-signal analysis improves confidence in risk narratives

Improvements Needed

  • • Refactor API queries out of the backend and onto the UI so non-technical PMs can access drive analysis
  • • Data ingestion limited due to cost measures where big data would yield better analysis
  • • The data ingestion engine and AI-Analysis is ready for scaling, but currently turned off
